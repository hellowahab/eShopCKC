## 🚀 Tech Stack

- **Backend:** ASP.NET Core 9
- **ORM:** Entity Framework Core
- **Database:** SQL Server
- **Architecture:** Clean Architecture with Repository Pattern
- **API:** RESTful APIs with proper layering and dependency injection
